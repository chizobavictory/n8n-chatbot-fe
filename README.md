# n8n GenAI Chatbot Frontend

A modern, full-screen React chatbot interface designed for the n8n GenAI Chatbot. Built with a focus on premium aesthetics and smooth user experience.

## ‚ú® Key UI Features

- **Glassmorphism Aesthetics**: Modern backdrop blurs and semi-transparent message bubbles.
- **Dark/Light/System Theme**: Intelligent theme switching with persistent user preference.
- **Smart Scroll**: Auto-scroll detection that respects manual reading and forces scroll on new messages.
- **Requirements Panel**: Integrated side panel with task checklist and full instructions.
- **Multimedia Support**: Seamless handling of text, images, and audio messages.
- **Premium Icons**: High-visibility action buttons for GitHub access, architecture diagrams, and requirements toggling.
